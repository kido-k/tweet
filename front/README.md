# tweet-analyze

> This is analyzer of twitter

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# generate static project
$ npm run generate
```

```bash
$ docker-compose up -d
$ docker-compose exec node sh
### ここからnodeコンテン内で実行
/app # npm install
/app # npm run dev
###　ここまで

↑を実行後、localhost:3000にアクセス
```
